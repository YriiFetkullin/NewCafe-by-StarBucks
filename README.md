NewCafe by Starbucks

Современный адаптивный лендинг для открытия нового кафе Starbucks.
Проект выполнен на чистом HTML5, CSS3 без использования фреймворков и библиотек.

О проекте

Данный проект представляет собой промо-сайт нового кафе Starbucks с акцентом на:

* современный UI/UX;
* адаптивность;
* чистую и понятную верстку;
* плавные анимации;
* удобство просмотра на любых устройствах.

Сайт разработан как showcase-проект для демонстрации frontend-навыков.

Используемые технологии

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries

Адаптивность

Сайт полностью адаптирован под:

* Desktop
* Mobile
* Tablet

Корректно отображается на современных браузерах:

* Google Chrome
* Safari
* Firefox
* Edge
* Opera

Реализованный функционал

* Адаптивное меню
* Hero-секция
* Блоки с продукцией
* Секции преимуществ
* Кастомные кнопки и hover-эффекты
* Плавные анимации появления
* Responsive layout
* Оптимизированная структура проекта

Цели проекта

* Практика адаптивной верстки
* Улучшение навыков работы с Flexbox/Grid
* Создание современного UI без библиотек
* Отработка чистой структуры HTML/CSS

Preview

Проект выполнен в стилистике Starbucks с акцентом на минимализм, чистый интерфейс и современную подачу контента.
